<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```bash
git clone https://github.com/Umar-Waseem/muhammadumarwaseem
```

<p>2. Install necessary packages</p>

```bash
npm install
```

<p>3. Start the local dev server</p>

```bash
npm run dev
```
